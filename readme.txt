Readme file:Prerequisites: You need following:* Python* Jupyter Notebook* Python libraries (numpy, pandas, matplotlib)Running the Notebook:To run the IPython notebook , follow these steps:1. Open terminal or command prompt.2. Navigate to the directory where the notebook file is located.3. Start Jupyter notebook server by running:-jupyter notebook4. In your web browser thr Jupyter note book interface should open automatically. Click on the notebook file (.ipynb) to open it.5. Inside notebook , click on rub button.
